# Sengoku-Dynasty-Thai
แปลไปเรื่อย

## วิธีการติดตั้ง
1. เปิดโฟลเดอร์เกม steamapps\common\Sengoku Dynasty\SengokuDynasty\Content\Paks
2. สร้างโฟลเดอร์ ~mods 
2. นำไฟล์ zSDThai_P.pak วางไว้ใน ~mods

## วิธีถอนการติดตั้ง
1. ลบไฟล์ zSDThai_P.pak ที่ไว้ใน ~mods
